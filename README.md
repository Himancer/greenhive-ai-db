# GreenHive.ai Database Engine
> Next.js 14 plant catalog with dynamic filters, rule-based chatbot, Prisma, and PostgreSQL.

## 🎯 What it does

This repository contains the source code for GreenHive.ai Database Engine. Next.js 14 plant catalog with dynamic filters, rule-based chatbot, Prisma, and PostgreSQL. It is designed to be scalable, efficient, and user-friendly.

## ✨ Features

- ⚡ **Core Functionality** — Reliable and robust execution.
- 🎨 **User Experience** — Designed with a focus on simplicity and ease of use.
- 🛠️ **Scalable Architecture** — Built to handle future growth and feature expansions.
